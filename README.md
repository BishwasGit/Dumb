A dreamy, interactive, mobile-first surprise website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Animated Lotus**: A central, organic lotus flower that blooms on load.
- **Floating Garden**: Floating Hearts, Stars, and Envelopes that you can interact with.
  - **Envelopes**: Reveal cute/funny quotes when clicked.
  - **Magic Pops**: Other items turn into flowers (Daisy, Tulip, Sakura) when tapped.
- **Angry Runchey Section**: A playful Q&A section at the top.
  - "Yes" triggers a shower of "Sorry" messages.
  - "No" triggers a cheeky response.
- **Audio**: Toggle for soft background music (placeholder).

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm start
    ```
    (or `npm run dev`)

3.  **Build for production**:
    ```bash
    npm run build
    ```
